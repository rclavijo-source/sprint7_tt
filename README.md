# sprint7\_tt

Proyecto Sprint 7 TripleTen



La aplicación web toma información de un conjunto de datos de ventas de coches y hace dos gráficos, un histograma donde se ve la distribución del uso de los coches en venta haciendo uso de la información del odómetro. También hace un gráfico de dispersión que compara el precio de los vehículos en venta contra la información del odómetro.



Al acceder a la aplicación el usuario pueda hacer uso de casillas de verificación para decidir si quiere ver solo una de las gráficas, o las dos.
